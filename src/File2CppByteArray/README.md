## File2CppByteArray.csx

Converts a file to a C++ bytes array.

#### Source code

* [File2CppByteArray.csx](File2CppByteArray.csx)

#### Usage

```
File2CppByteArray.csx <file-name>
```

#### Example output

```
File2CppByteArray.csx smallLogo.jpg
```
