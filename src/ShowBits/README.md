## ShowBits.csx

Shows bits of any number.

#### Source code

* [ShowBits.csx](ShowBits.csx)

#### Usage

```
ShowBits.csx <dec-or-hex-number>
```

#### Example output

```
ShowBits.csx 0x5678
```

```
Bits of    0x5678 or 22136:
 * Bit  3: 0x0008 or 8
 * Bit  4: 0x0010 or 16
 * Bit  5: 0x0020 or 32
 * Bit  6: 0x0040 or 64
 * Bit  9: 0x0200 or 512
 * Bit 10: 0x0400 or 1024
 * Bit 12: 0x1000 or 4096
 * Bit 14: 0x4000 or 16384
```
